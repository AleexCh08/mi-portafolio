# 🚀 Mi Portafolio

Un portafolio web interactivo y de alto rendimiento construido para demostrar mis capacidades como Desarrollador de Software. Diseñado con un enfoque *mobile-first*, animaciones fluidas y una arquitectura de componentes escalable.

[![Ver Portafolio](https://img.shields.io/badge/🌐_Ver_Portafolio-blue?style=for-the-badge&logoColor=white&color=2563EB)](https://mi-portafolio-aleexch.netlify.app/)

> **🎯 Nota para Reclutadores y Líderes Técnicos:**  
> La mejor manera de experimentar este proyecto es visitando la versión desplegada en el enlace de arriba. El código fuente está disponible a continuación para revisión técnica.

## ✨ Características Técnicas Destacadas

Este portafolio no es una plantilla estática. Fue desarrollado desde cero implementando soluciones técnicas modernas:

*   **Carrusel 3D Interactivo:** Visualización de proyectos mediante un efecto *Coverflow* calculado matemáticamente, con soporte para navegación por botones y puntos de control.
*   **Soporte de Gestos Nativos:** Integración de deslizamiento táctil (*swipe*) fluido para dispositivos móviles utilizando físicas de rebote y restricciones de arrastre.
*   **Modales Dinámicos (Sistema Cinematográfico):** Ventanas emergentes optimizadas para reproducir demostraciones en formato de video (`.mp4`/`.webm`) sin afectar el tiempo de carga inicial de la página (LCP).
*   **UI/UX Reactiva:** Interfaz responsiva gestionada con Tailwind CSS, incluyendo efectos de resplandor (*glow*), sombras dinámicas y retroalimentación táctil (`whileTap` / `whileHover`).

## 📁 Proyectos Destacados en el Portafolio

En la sección de proyectos encontrarás software desarrollado en diversos entornos y lenguajes:

1.  **Rey de Diamantes (CDB):** PWA Mobile-first construida con React y Vite.
2.  **OBJ Viewer:** Visor de modelos 3D en tiempo real usando C++, OpenGL y GLFW.
3.  **Gestión de Loterías:** Plataforma web full-stack con Django y Web Scraping.
4.  **PhotoLab Pro:** Procesamiento Digital de Imágenes (PDI) en Kotlin y JavaFX.
5.  **Dredge Clone:** Motor de simulación y físicas marinas utilizando C++ y Raylib.

## ⚙️ Instalación y Ejecución Local

Si deseas correr este portafolio en tu máquina local para revisar el código fuente o el entorno de desarrollo:

1. Clona el repositorio:
   ```bash
   git clone [https://github.com/AleexCh08/mi-portafolio.git]
   ```

2. Navega al directorio del proyecto:
    ```bash
    cd nombre-de-tu-repo
    ```

3. Instala las dependencias:
    ```bash
    npm install
    ```

4. Inicia el servidor de desarrollo local:
    ```bash
    npm run dev
    ```
